Za analizo podatkov sem se odločil analizirati filme. Zanima me predvsem, ali so se filmi "poslabšali" torej, da se je njihova povprečna ocena znižala v zadnjih letih, ali filmi postajajo daljši ali krajši in katere vrste filmov so najbolje ocenjenje. Podatke sem zbiral iz spletne strani https://www.imdb.com/. Pomeni, da so podatki posodobljeni, ne glede na to, katero obdobje si izberemo za analizo. Program prenese podatke kot so naslov, ocena, trajanje in leto izzida. Te podatke potem shrani v .txt datotekah in jih potem predela, ter za vsako izbrano vrsto filmov izriše grafe.
